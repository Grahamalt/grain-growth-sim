# grain-growth-sim

Potts-model Monte Carlo simulation of grain growth with solute segregation and diffusion, plus an optical scattering proxy for linking microstructure to optical response.

## Structure

- `code/` — simulation, analysis, and plotting modules
- `tests/` — unit tests
- `results/` — generated figures and data
- `docs/` — theory notes and problem statement

## Quick start

```bash
python code/main.py
```

See `problem_statement.md` for the scientific problem and `CLAUDE.md` for session context when working with Claude Code.
